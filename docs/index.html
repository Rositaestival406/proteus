Core Features

High Performance

Rust-powered core engine with parallel processing via Rayon. Analyze files in under 50ms with multi-threaded batch processing capabilities.

Deep Binary Analysis

Advanced PE and ELF parsing using goblin library. Extract imports, exports, sections, and metadata with precision and accuracy.

Machine Learning Ready

Extract 16+ features per sample for ML pipelines. Includes Random Forest (96% accuracy) and Isolation Forest classifiers with anomaly detection.

Entropy Calculation

Shannon entropy analysis at both global and section levels. Detect packed, encrypted, or obfuscated malware with high accuracy.

String Analysis

Extract ASCII and wide strings with pattern detection. Identify URLs, IPs, registry keys, file paths, and suspicious keywords automatically.

YARA Rule Engine

Industry-standard malware detection with 40+ pre-built rules. Detect ransomware, RATs, trojans, packers, and suspicious behaviors with high precision.

Multi-Layer Detection Engines

YARA Rule Engine (40+ Rules)

  • Ransomware Detection: WannaCry, Ryuk, Maze, Locky, Generic patterns
  • RAT Detection: NanoCore, njRAT, DarkComet, Quasar, AsyncRAT
  • Banking Trojans: Emotet, TrickBot, Dridex, Zeus, Formbook, AgentTesla
  • Packer Detection: UPX, ASPack, Themida, VMProtect, PECompact, MPRESS
  • Suspicious Behaviors: Code injection, process hollowing, anti-VM, credential dumping, keyloggers

Machine Learning Detection

  • Random Forest Classifier: 96.22% accuracy on real-world dataset
  • Isolation Forest: Anomaly detection for unknown threats
  • Feature Extraction: 16+ features per sample
  • Training Dataset: 1,190 samples (576 malware + 614 clean)

Heuristic Analysis

  • Entropy Analysis: Detect packed/encrypted binaries
  • API Monitoring: Suspicious Windows/Linux API usage
  • String Analysis: IOC extraction (URLs, IPs, registry keys)
  • Section Analysis: Abnormal PE/ELF section characteristics

Quick Installation

# Clone the repository
git clone https://github.com/ChronoCoders/proteus.git
cd proteus

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Install dependencies
pip install -r requirements.txt

# Build Rust module
maturin develop --release

# Analyze a file (all engines)
python cli.py file sample.exe --ml --yara --strings

# Collect malware dataset
python malware_collector.py
